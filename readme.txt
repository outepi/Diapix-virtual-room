Audio3D available at: https://www.phon.ucl.ac.uk/resource/audio3d/ (by Mark Huckvale)

version 1.81 (or above)

Configuration (text) files set separately for each "rooms" that contain:
3 x noise source (child, male, female) that have environmental sound detection task embedded in them
1 x live source (the other talker)

Levels set and calibrated using TotalMix and Fireface audio interface with Beyerdynamic DT297 headphones. 

Example files contain one picture pair (Beach, version 1)
For speech masker audio: Related (B1), Unrelated (S1)
